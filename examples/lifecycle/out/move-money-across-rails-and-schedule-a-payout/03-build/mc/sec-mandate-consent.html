Capture and display a debit authorization (ACH authorization / SEPA mandate): payee, amount cap, frequency, and a revocation path.

When to use

Setting up any pull-based recurring debit where the payer must authorize in advance.

When not to / use instead

Push payments the user initiates each time — there is no standing mandate to capture.

Behaviour & accessibility contract

The authorization text is readable before consent; revoke is as easy to find as grant; consent state is announced.

Regulatory anchor

NACHA authorization rules (WEB / PPD / CCD) · SEPA Direct Debit Mandate (Rulebook) · Reg E preauthorized transfers (12 CFR 1005.10)
