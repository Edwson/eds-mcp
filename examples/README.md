# Examples

- **`library-usage.mjs`** — use eds-mcp as a plain Node library (no MCP). `node examples/library-usage.mjs`
- **`claude_desktop_config.json`** — drop the `mcpServers.eds` block into your MCP client config
  (Claude Desktop / Claude Code / Cursor `mcp.json`). Set `cwd` to the absolute path of this repo.
